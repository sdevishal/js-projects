# ⏱️ Digital Stopwatch

A simple digital stopwatch built with **HTML, CSS, and JavaScript**.

## 🚀 Features
- Start, pause, and reset functionality.
- Displays time in **HH:MM:SS** format.
- Responsive design (works on mobile/desktop).

## 🛠️ Tech Stack
- **HTML5** (Structure)
- **CSS3** (Styling)
- **JavaScript** (Logic)
